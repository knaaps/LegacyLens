# LegacyLens Source

This directory acts as the main package directory for `legacylens`.

## Contents
- **`legacylens/`**: The core Python package housing the main modules:
  - Code parsers and AST extractors
  - Vector database and retrieval integrations (RAG)
  - The Prompt Generator and Generator LLM handlers
  - The Critic model (verification loops)
  - System orchestration
- **`legacylens.egg-info/`**: Package metadata generated during installation (e.g., via `pip install -e .`).
