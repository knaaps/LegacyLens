# LegacyLens Dashboard

This directory contains the React-based web dashboard for LegacyLens. It provides a web interface to explore, search, and visualize code explanations, function metadata, and verified RAG outputs.

## Tech Stack
- React + TypeScript
- Vite (bundler and dev server)
- Tailwind CSS

## Getting Started

To run the dashboard locally for development:

```bash
cd dashboard
npm install
npm run dev
```

The interface allows viewing the interactive scatter plots, explanation modals, and search functionality as generated by the LegacyLens pipeline.
